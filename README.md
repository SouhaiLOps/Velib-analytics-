# Vélib' analytics – EDA & collecte

Notebooks d’exploration sur les données Vélib’ (GBFS / archives), carte Folium,
et scripts de snapshot.

## Structure
- `notebooks/` : EDA (historique, carto, clustering…)
- `scripts/` : ex. `snapshot_velib.py`
- `data/processed/` : résultats légers (parquet) – **optionnel**
- `data/raw/`, `data/external/` : **ignorés** par Git
- `.venv/` : environnement Python local, **ignoré**
